◆ oneFileGraph()

Graph ReadCSV::oneFileGraph ( )

Constructs a graph from a single CSV file. The oneFileGraph() function constructs a graph by reading data from a single CSV file. The CSV file should contain lines representing edges between vertices, with each line containing the source vertex ID, destination vertex ID, and distance. If the CSV file includes additional columns, it considers them as labels for the vertices.

Note
The time complexity of this function depends on the size of the CSV file and the number of vertices and edges in the graph.
Returns
The constructed graph.
